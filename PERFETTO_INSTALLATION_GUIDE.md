# Perfetto Trace Analysis - Installation & Setup Guide

## Quick Start

### 1. Install Python Package

```bash
pip install perfetto-tools
```

### 2. Verify Installation

```bash
python -c "from perfetto.trace_processor import TraceProcessor; print('OK')"
```

### 3. Test Preprocessing

```bash
python ./.claude/skills/perfetto-trace-analysis/scripts/preprocess.py \
  --input <your-trace.pftrace> \
  --target-fps 30 \
  --output-dir ./output/perfetto
```

## Installation

### Prerequisites

- Python 3.6+
- pip package manager
- Windows/macOS/Linux terminal

### Step 1: Install perfetto-tools

```bash
pip install perfetto-tools
```

### Step 2: Verify Files Exist

Check these files are present:
- `./tools/trace_processor_shell.exe`
- `./.claude/skills/perfetto-trace-analysis/scripts/preprocess.py`
- `./.claude/skills/perfetto-trace-analysis/config.json`

---

## Usage Examples

### Basic Usage
```bash
python ./.claude/skills/perfetto-trace-analysis/scripts/preprocess.py \
  --input recording.pftrace \
  --target-fps 30
```

### With Custom Output
```bash
python ./.claude/skills/perfetto-trace-analysis/scripts/preprocess.py \
  --input recording.pftrace \
  --target-fps 30 \
  --output-dir ./my-output
```

### With Custom Config
```bash
python ./.claude/skills/perfetto-trace-analysis/scripts/preprocess.py \
  --input recording.pftrace \
  --config ./my-config.json
```

## Output

Generated files go to: `./output/perfetto/`

- `preprocess-result.json` - Structured analysis data
- `perfetto-report_*.md` - Final analysis report (generated by Claude skill)

---

## Troubleshooting

### ModuleNotFoundError: No module named 'perfetto'

```bash
pip install perfetto-tools
```

### File not found: preprocess.py

Make sure you're in the project root directory, or use full path.

### Invalid .pftrace file

Verify the file is a valid Perfetto trace using:
```bash
file recording.pftrace
```

---

## Configuration

Edit `./.claude/skills/perfetto-trace-analysis/config.json` to customize:

- `gameProcess` - Your app package name
- `targetFps` - Target frame rate (default: 30)
- `mainThread` - Main thread name (default: UnityMain)
- `renderThread` - Render thread name (default: UnityGfxRenderS)

---

## Integration with Claude

Once preprocessing is complete, invoke the perfetto-trace-analysis skill:

```
User: Analyze this perfetto trace recording.pftrace for performance issues
```

Claude will automatically:
1. Read the preprocess-result.json
2. Perform 8-dimension analysis
3. Generate detailed report
